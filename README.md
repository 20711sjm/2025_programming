# 체대 입시 기록 관리 및 영상 피드백 시스템

체대 입시생을 위한 종합적인 기록 관리 및 영상 피드백 시스템입니다.

## 주요 기능

### 1. 📊 기록 입력
- 운동 기록 입력 및 저장
- 날짜, 시간대, 날씨, 컨디션 등 상세 정보 기록
- 메모 기능

### 2. 📈 기록 비교 및 분석
- 종목별 기록 추이 시각화
- 개인 최고 기록(PB) 표시
- 향상률 계산 및 표시
- 기록 상세 내역 조회

### 3. 🎥 영상 관리
- 훈련 영상 업로드 (MP4, MOV, AVI)
- 영상 메타데이터 관리
- 영상 목록 조회 및 재생

### 4. 💬 피드백
- 코치 피드백 작성 및 저장
- 영상 시간대별 피드백
- 피드백 유형 분류 (전체 평가, 기술 지적, 개선 사항 등)
- 기존 피드백 조회

### 5. 📄 리포트
- 목표 기록 설정
- 목표 달성률 계산
- PDF 리포트 생성 및 다운로드

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 애플리케이션 실행:
```bash
streamlit run app.py
```

## 데이터 저장 구조

- `data/records.json`: 운동 기록 데이터
- `data/goals.json`: 목표 기록 데이터
- `data/videos_metadata.json`: 영상 메타데이터
- `data/feedback.json`: 피드백 데이터
- `data/videos/`: 업로드된 영상 파일
- `data/report_*.pdf`: 생성된 리포트 파일

## 사용 방법

1. **기록 입력**: 사이드바에서 "📊 기록 입력" 메뉴를 선택하여 운동 기록을 입력합니다.

2. **기록 분석**: "📈 기록 비교 및 분석" 메뉴에서 종목별 기록 추이를 확인하고 PB와 향상률을 확인합니다.

3. **영상 관리**: "🎥 영상 관리" 메뉴에서 훈련 영상을 업로드하고 관리합니다.

4. **피드백**: "💬 피드백" 메뉴에서 코치가 영상에 대한 피드백을 작성합니다.

5. **리포트**: "📄 리포트" 메뉴에서 목표를 설정하고 달성률 리포트를 생성합니다.

## 기술 스택

- **Python**: 프로그래밍 언어
- **Streamlit**: 웹 프레임워크
- **Pandas**: 데이터 처리
- **Plotly**: 시각화
- **ReportLab**: PDF 생성

## 향후 개선 사항

- OpenAI API를 활용한 자동 피드백 생성
- MediaPipe를 활용한 자동 동작 분석
- Google Cloud Vision을 활용한 이미지 분석
- 클라우드 저장소 연동 (Firebase 등)
- 사용자 인증 및 권한 관리
- 다중 사용자 지원
